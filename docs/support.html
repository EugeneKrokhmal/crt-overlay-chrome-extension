CRT Monitor Overlay

Chrome extension – Support, usage, and privacy

What it does

CRT Monitor Overlay adds a retro CRT-style visual effect on web pages: scanlines, vignette, screen curvature, and optional VHS-style sound (muffled audio + light hiss) for in-page video and audio. The overlay does not block clicks or keyboard—you can use the page normally.

  • Toggle overlay on or off from the extension popup or with Ctrl+Shift+C (Windows/Linux) or Command+Shift+C (Mac).
  • Adjust effect strength with sliders (scanlines, vignette, curvature, glow, VHS sound, and glitches).
  • Settings are saved locally and apply to all tabs.

How to use

  1. Click the extension icon in the Chrome toolbar to open the popup.
  2. Turn Enable overlay on. Use the sliders to tune the effect.
  3. Optionally enable VHS sound and VHS glitches for a more retro look and sound.
  4. Use the keyboard shortcut to toggle the overlay without opening the popup.

Privacy policy

CRT Monitor Overlay does not collect, store on external servers, or transmit any personal data.

  • Your preferences (e.g. overlay on/off, slider values) are stored only on your device using Chrome’s local storage (chrome.storage.local).
  • No analytics, no tracking, no account, no server. The extension does not send data to the developer or any third party.
  • Audio processing (VHS sound) is done in your browser only and is not recorded or sent anywhere.

This policy applies to the extension as distributed on the Chrome Web Store. If you obtained the extension from another source, that source may have different practices.

Why these permissions?

Chrome asks for the following. Here is how CRT Monitor Overlay uses them:

  • Storage – To save your settings (on/off, slider values) on your device so they persist across sessions and tabs.
  • Active tab – To know which tab is active when you open the popup, so the extension can work correctly.
  • Tabs – To apply your overlay and settings to all open tabs when you change them, and to keep new tabs in sync.

No permission is used to read your browsing history, track you, or send data off your device.

FAQ

Does it work on every site? The overlay can run on any page. The VHS sound effect only affects <audio> and <video> that the page loads directly; it does not change audio inside iframes (e.g. many embedded YouTube players) due to browser security.

Can I turn it off? Yes. Use the popup to disable the overlay, or the same keyboard shortcut to toggle. Your last setting is remembered.

Is it open source? If the extension is published with a linked source or repository, you can inspect the code there.
